Projects

Projects

Below are the sample of academic projects in Data Analytics using SQL, Python, DIP, Blockchain and Machine Learning, demonstrating practical applications from coursework.

Fire security alarm & automatic water flow, sending location with GPS control system

A smart fire detection & response system is designed for homes. Upon detecting fire, the system activates an alarm and immediately disperses water to extinguish it. If the fire persists, pressing the emergency button sends the house’s location to the nearest fire station and makes an automated call.

UIU Bookshelf as like as Vending machine

Developed a smart bookshelf resembling a vending machine, integrated with mobile connectivity. Users can select books via an app, scan RFID for confirmation and retrieve chosen books from shelves. Automated email notifications provide book details and return dates, enhancing user convenience & system efficiency.

UIU Art Gallery system

Developed a comprehensive software solution for art gallery exhibitions featuring separate portals for admins, users & artists. Users can purchase various art pieces like sketches, craft and paintings etc. Also share feedback through star ratings & comments. Enhances gallery interaction and sales through intuitive online platforms.

UIU Club Management System

Developed an online platform for clubs to streamline member management and enhance communication. It allows users to create profiles, join clubs, view events & engage in real-time chat. It also supports event planning, payment processing & leadership elections, improving the efficiency & connectivity of club activities. We develop this project via Bootstrap, Figma, Jira.

MedHelp for Healthcare Service

Developed "MedHelp," a comprehensive healthcare software enabling users to book doctor appointments, order medicines, schedule medical tests and request emergency ambulance services with real-time tracking. Enhanced patient doctor communication, reduced wait times, and improved accessibility to medical services. Implemented secure user authentication, responsive design and multilingual support

UIU Event Management

Developed a comprehensive web application, "Capital Event Management," for online event bookings like weddings & parties. Utilizing HTML, CSS, JS, PHP, and MySQL, the system streamlines event management with features for booking, venue selection, bill calculation, and user feedback. The user-friendly interface supports both novice and advanced users, ensuring efficient booking & management.

Deep Learning for Banana Leaf Disease Detection — ML Based

Developed a CNN-based approach to diagnose banana leaf diseases like Fusarium Wilt and Black Sigatoka using machine learning models. Achieved high accuracy with VGG16 (94%), ResNet50v2 (95.55%) & MobileNet (97%) models. This project demonstrates the potential of AI in enhancing agricultural productivity and disease management.

ScRNA-seq dropout imputation using temporal dependencies & graph attention network

Developed a novel approach using Graph Convolutional Networks (GCN) & Graph Attention Networks (GAN) within an autoencoder framework to improve dropout imputation in single-cell RNA sequencing (scRNA-seq). Achieved 89% accuracy on benchmark datasets, demonstrating robustness through Normalized Mutual Information (NMI) and Adjusted Rand Index (ARI). This method enhances insights into cellular heterogeneity, crucial for precise biological studies.

Soccer player Injury Prediction Using sequence Matching — Data Mining Based Project

Developed a predictive model for soccer player injuries using data from 2,500+ professionals, employing K-means clustering & separate RNNs for distinct injury subgroups. Evaluated with precision metrics, it enables personalized injury risk assessment and prevention strategies in professional football, enhancing player health and career longevity.

Cross-Transaction Network for Mobile Financial Services Using Hyperledger Fabric — Computer Security Based Project

Implemented Hyperledger Fabric blockchain to connect Mobile Financial Service (MFS) providers in Bangladesh, enhancing security and transparency for cross-provider transactions. This initiative supports the rapid adoption of digital payments and strengthens the MFS ecosystem amidst growing financial mobilization.

IoT- based Green Infrastructure for Healthcare Ecosystem — Green Computing Based Project

Implemented green healthcare practices at our hospital to enhance patient comfort, energy efficiency & cost savings. Utilized sensor based temperature management, smart HVAC systems & automated lighting controls to optimize energy usage. Added tree planting for natural temperature regulation, enhancing patient comfort, reducing system maintenance & setting a new standard in patient care.

SmartAgroChain: Revolutionizing Agricultural Supply Chains with Blockchain Technology — Final Year Design Project (Thesis)

Developed SmartAgroChain, a decentralized blockchain solution for agricultural supply chains ensuring fairness, transparency, and trust. It facilitates direct farmer-to-consumer transactions through community-driven auctions, eliminating middlemen and ensuring fair pricing. Implemented on Ethereum, it includes escrow mechanisms for secure transactions, showing promising scalability and acceptance in rural markets.
